Current Models and Text Services

1. Heuristic Rule-Based Entity Tagger

Uses database and a rule based approach to identify medical and numeric entities.

API Routes
  • api/get_tags: Post request accepts a block of text and returns "Spacy Formatted" entities.
  • api/drugs_and_conditions: Post request accepts a block of text and returns a list of unique drugs and medical conditions contained in the text.

    API Routes
